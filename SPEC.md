# 子どもの学習スタイル診断【無料】無料ゲーム - 技術仕様書

## 概要

**サービス名**: Kids Learning Style Quiz
**バージョン**: 1.1.0
**更新日**: 2026-05-29
**URL**: https://appadaycreator.com/kids-learning-style-quiz/

子どもの得意なこと・好み・行動パターンから最適な学習スタイルと習い事を診断。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 「診断スタート ▶」ボタンをタップする
2. Q1〜Q5の質問を読み、当てはまる選択肢をタップ
3. 5問回答すると学習スタイルタイプが自動表示（5タイプのスコアバー付き）
4. おすすめ習い事・効果的な学習法を確認する
5. 「印刷」「コピー」ボタンで結果を保存・共有

| タイプ | 特性 | おすすめ習い事 |
|-------|------|--------------|
| 🔭 論理・数学的知能 | 数・論理への強い関心 | プログラミング、ロボット教室 |
| 🤸 身体・運動 | 体を動かして学ぶ | スポーツ全般、体操 |
| 🎨 視覚・空間 | 図・絵から取り込む | 絵画教室、工作 |
| 🎵 音楽・リズム | リズム・音の記憶力 | ピアノ、バイオリン |
| 📖 言語・文章 | 言葉への感受性 | 読書、英会話 |

## よくある質問（FAQ）

**Q: 子どもの学習スタイル診断は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
